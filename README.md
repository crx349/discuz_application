## 说明

分享日常Discuz!开发中的精品应用。

## 应用列表

* find_username （通过邮箱找回用户名，适合站点运营较久（多个100W+用户站点使用中），用户一般自动登陆，容易忘记用户名，可以通过邮箱找回）

## 主要特性

* 应用完全免费
   
* 支持gbk和utf8版本
  
## 兼容性说明

理论兼容Discuz! X2.5-X3.5

## 安装使用

source 目录是插件目录

template 目录是模板目录 

请根据需要上传指定目录到网站根目录，后台-应用-未安装应用-安装
